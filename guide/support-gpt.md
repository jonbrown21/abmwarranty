<p>
The ABM Warranty Support GPT is designed to assist administrators in
troubleshooting configuration and import issues by interpreting log output
and explaining expected application behavior.
</p>

<p>
The GPT understands ABM Warranty’s authentication flow, import lifecycle,
coverage logic, retry behavior, and throttling rules. It can help explain
errors related to credentials, PEM keys, token exchange, API responses, and
coverage retrieval.
</p>

<p>
For best results, export relevant log entries from the Log window and include
them when requesting assistance. Providing additional context—such as whether
an import was canceled, throttled, or run in diagnostic mode—can further
improve accuracy.
</p>

<p>
The Support GPT does not replace the Status Dashboard or Logs. Instead, it
serves as a complementary tool to help interpret diagnostic output and guide
next steps when issues are unclear.
</p>
