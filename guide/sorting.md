<p>
ABM Warranty supports sorting the device list by <strong>serial number</strong>
in either ascending (A→Z) or descending (Z→A) order. Sorting is applied after
filters, ensuring that the displayed device list remains stable and predictable.
</p>

<p>
The selected sort order affects:
</p>

<ul>
    <li>The device list shown in the sidebar</li>
    <li>Keyboard and navigation-based device browsing</li>
    <li>Dashboard-driven selections and transitions</li>
    <li>CSV exports generated from the current view</li>
</ul>

<p>
Sort order is a per-session presentation preference. It remains active while
navigating between devices, applying filters, or viewing device details, but
does not alter stored device data or import behavior.
</p>

<p>
Sorting operates independently per tenant. When switching tenants, the current
sort order is reapplied to that tenant’s device list without modifying its
underlying data.
</p>

<p>
Sorting does not impact import performance, API requests, or coverage evaluation.
It affects only how devices are displayed and exported.
</p>
